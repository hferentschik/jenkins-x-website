Merry Jenkins X Mas!

Merry Jenkins X Mas!

December 20, 2018

As we head towards the holiday season I wanted to call out some recent events in the Jenkins X community. Its been an amazing year and we’re looking forward to celebrating Jenkins X birthday in January!

Webinar on Serverless Jenkins and Jenkins X

Yesterday James Strachan and James Rawlings gave a webinar on Servlerless Jenkins with Jenkins X. You should be able to watch the recording (registration required).

We had hundreds of folks who asked some really great questions - thanks to all those who attended! A few of the highlighted questions:

  • Jenkins X has git support for multiple git providers like GitHub, GitHub Enterprise, BitBucket Cloud, BitBucket Server and GitLab when using static Jenkins masters - however currently Serverless Jenkins only supports GitHub. We’re working on it and hope to add more git providers soon!
  • Here is how you can avoid Tiller when using helm with Jenkins X

KubeCon was awesome

A bunch of us from CloudBees had a booth at KubeCon Seattle where we spent lots of time chatting with folks on Jenkins X, Serverless Jenkins and the new commercial support for Jenkins X.

There was a huge amount of excitement in both automating CI/CD for kubernetes and making Jenkins serverless. Thanks to all those who popped by for a chat!

Also Carlos did a great talk and live demo on Jenkins X: Continuous Delivery for Kubernetes

Office Hours

We had an Office Hours last week at KubeCon Seattle, you can check out the notes here and watch the video here

Please join the next one in January! Or if you can’t wait just jump onto slack
